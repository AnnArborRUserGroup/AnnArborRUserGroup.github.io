Ann Arbor R User Group

Welcome

The Ann Arbor R User Group is an open-source, community dedicated to sharing knowledge and experience in solving data analysis and statistical problems using R.

News

  • The June meetup, Interactive Web Applications in R with Shiny, was on June 4th. Clayton Yochum walked us through the steps to create, iterate on, and host a simple web application [slides].

  • The May meetup, R Markdown and Support Vector Machines, was on May 7th. Marian Schmidt gave an introduction to using R Markdown [slides] and Michelle Berry gave an overview of support vector machines [slides].

  • The April meetup, Classification: Regularized logistic regression, Trees and random forests, was on April 9th. Devin Riley demonstrated binary classification using regularized logistic regression [slides] and Ellis Valentiner gave an introduction to decision trees and random forests [slides]

  • The March meetup, Data Exploration, was on March 12. Kun Deng gave an introduction to the tidyr and dplyr packages [slides], Haad Khan showed us how to perform exploratory data analysis and visualization using ggplot2 [slides], and Blake Nicholson walked us through the basics of principal component analysis and exploratory factor analysis [slides]

  • The February meetup, Where do we go from here?, was on February 5. The group had an organizational meeting to plan Meetups for early 2015.
